A desktop application that analyzes WhatsApp chat data using Natural Language Processing (NLP).
It extracts important insights such as message frequency, active timelines, named entities, and trending topics through visualizations like graphs and word clouds.
Built using Python with a simple and user-friendly tkinter GUI.
